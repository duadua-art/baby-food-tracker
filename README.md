# 🥕 副食品日記

> 專為新手爸媽設計的寶寶副食品紀錄 App，記錄每一口的成長旅程。

**🌐 線上使用：[duadua-art.github.io/baby-food-tracker](https://duadua-art.github.io/baby-food-tracker/)**

---

## ✨ 功能特色

- **食材卡點一下即記錄** — 點選食材卡片，快速紀錄進食時間、份量與寶寶反應
- **喜好追蹤** — 記錄寶寶對每種食材的喜好（😍 喜歡 / 😐 普通 / 😤 不喜歡）
- **過敏反應紀錄** — 偵測到過敏症狀時自動啟動 3 天試敏倒數計時
- **副食品經驗值系統** — 嘗試越多食材，等級越高（Lv.1 美食小探險家 → Lv.10 副食品傳說）
- **我的食譜** — 記錄自創食譜，並可透過連結分享給家人朋友
- **家庭共同記錄** — 爸爸、媽媽、阿公、阿嬤輸入同一組家庭代碼，即可共享所有紀錄
- **多寶寶切換** — 支援第二個寶寶，資料完全獨立
- **日曆安排** — 專為Ｊ人爸媽設計，可以預先規劃一週食材，根據規劃採購與準備寶寶副食已嘗試
- **離線可用** — 安裝為 PWA 後，無網路也能正常記錄
- **AI食譜推薦** - (此功能有規劃但目前版本先拿掉了)等有需要再來更新😆

## 📱 安裝到手機主畫面（PWA）

### iPhone（Safari）
1. 用 **Safari** 開啟 [網站連結](https://duadua-art.github.io/baby-food-tracker/)
2. 點下方 **分享按鈕** □↑
3. 選 **加入主畫面**
4. 命名為「副食品日記」→ 新增

### Android（Chrome）
1. 用 **Chrome** 開啟網站連結
2. 點右上角 **⋮**
3. 選 **安裝應用程式** 或 **新增至主畫面**

## 🚀 使用指南

### 基本設定
點右上角頭像進入**家庭設定**，設定你的角色（爸爸／媽媽／阿公⋯⋯）與暱稱。點左上角寶寶名稱可**新增或切換寶寶**，設定寶寶暱稱與出生日期。

### 記錄副食品
1. 切換到**食材**頁面
2. 點選任一食材卡片
3. 在底部彈出視窗選擇喜好、份量，如有異狀開啟過敏開關並選擇反應
4. 點**完成紀錄**

### 過敏倒數
記錄到過敏反應後，**今日**頁面會自動出現試敏倒數卡，顯示距離可重新嘗試還有幾天。

### 新增食譜
切換到**食譜**頁面，點右上角「＋ 新增食譜」，輸入食譜名稱與做法（最多 500 字），完成後可點**分享食譜**，將食譜連結傳給家人或朋友。

### 家庭共用
在**家庭設定**中點「建立新家庭」產生 6 位代碼，將代碼告訴其他家人，所有人輸入相同代碼後即可共同查看紀錄。
自動同步，同步更新時間大約為30秒，如果等不及也可以手動點選同步👌🏻
