# 🥕 副食品日記

> 專為新手爸媽設計的寶寶副食品紀錄 App，記錄每一口的成長旅程。

**🌐 線上使用：[duadua-art.github.io/baby-food-tracker](https://duadua-art.github.io/baby-food-tracker/)**

---

## ✨ 功能特色

- **食材卡點一下即記錄** — 點選食材卡片，快速紀錄進食時間、份量與寶寶反應
- **喜好追蹤** — 記錄寶寶對每種食材的喜好（😍 喜歡 / 😤 不喜歡）
- **過敏反應紀錄** — 偵測到過敏症狀時自動啟動 3 天試敏倒數計時
- **行事曆備忘** — 專為Ｊ人爸媽設計，預先規劃一週菜單，點選日期即可新增飲食計畫
- **副食品經驗值系統** — 嘗試越多食材，等級越高（Lv.1 美食小探險家 → Lv.10 副食品傳說）
- **我的食譜** — 記錄自創食譜，並可透過連結分享給家人朋友
- **家庭即時同步** — 爸爸、媽媽、阿公、阿嬤輸入同一組家庭代碼，搭配 Firebase 即可跨裝置即時共享所有紀錄
- **多寶寶切換** — 支援多個寶寶，資料完全獨立，可上傳寶寶大頭貼
- **滑動刪除紀錄** — 在紀錄頁向左滑動即可刪除單筆紀錄
- **離線可用** — 安裝為 PWA 後，無網路也能正常記錄
- **暫時移除AI推薦功能** - 等有人需要再來更新😆

## 📱 安裝到手機主畫面（PWA）

### iPhone（Safari）
1. 用 **Safari** 開啟 [網站連結](https://duadua-art.github.io/baby-food-tracker/)
2. 點下方 **分享按鈕** □↑
3. 選 **加入主畫面**
4. 命名為「副食品日記」→ 新增

### Android（Chrome）
1. 用 **Chrome** 開啟網站連結
2. 點右上角 **⋮**
3. 選 **安裝應用程式** 或 **新增至主畫面**

## 🚀 使用指南

### 第一步：基本設定
1. 點右上角**頭像**進入家庭設定
2. 選擇你的角色（爸爸／媽媽／阿公⋯⋯）並填入暱稱
3. 點左上角寶寶名稱可**新增或切換寶寶**，並可上傳寶寶大頭貼

### 記錄副食品
1. 切換到**食材**頁面，選擇分類（蔬菜／穀物／蛋白質／水果）
2. 點選任一食材卡片
3. 在底部彈出視窗選擇喜好、份量
4. 如有異狀，開啟過敏開關並選擇反應症狀
5. 點**完成紀錄**

> 食材卡片顏色說明：
> - 😲 灰色 — 尚未嘗試
> - 😍 / 😤 綠色 — 已嘗試（喜歡／不喜歡）
> - 😍 / 😤 紅色 — 已嘗試但有過敏反應

### 刪除紀錄
在**紀錄**頁面，對任一筆紀錄**向左滑動**，即可出現刪除按鈕。

### 行事曆備忘
切換到 **📅 行事曆**，點選任一日期可新增當天的飲食計畫或備忘文字，有備忘的日期會顯示橘色圓點。

### 新增食譜
切換到**食譜**頁面，點右上角「＋ 新增食譜」，輸入食譜名稱與做法（最多 500 字），完成後可點**分享食譜**將連結傳給家人。

### 家庭共用（同一支手機）
1. 進入**家庭設定**
2. 點「建立新家庭」產生 6 位代碼
3. 將代碼告訴其他家人，所有人輸入相同代碼即可共享紀錄
